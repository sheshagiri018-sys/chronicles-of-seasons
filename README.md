## 🌟 About the Project

🌈 **Chronicles of Seasons** is not just a website — it is a **visual emotional journey**.

It represents:
- ☀ Happiness → Summer
- 🌧 Sadness → Rain
- 🍂 Change → Autumn
- ❄ Silence → Winter
- 🌸 Rebirth → Spring
- 🌌 Infinity → Cosmos (Hidden Mode)

Every scroll feels like moving through **life itself**.

## 🎬 Experience Flow


🧠 Mood Selection → 🌌 Loader → 🌦 Seasons Journey → 🌍 Finale → ✨ Cosmos Unlock
